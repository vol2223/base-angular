# Base Angularjs
use the library [Angularjs](https://angularjs.org/) + [typeScript](http://www.typescriptlang.org/) + [browserify](http://browserify.org/)

# Install

### Quick build

**In the case of OSX**

```sh
$ git submodule init
$ git submodule update

$ brew install node

$ npm install coffee-script -g
$ npm install coffee-script

$ npm install gulp -g
$ npm install gulp

$ gulp
```
